Harness Runner

Supervisor Agent

任务拆解、激活子 Agent、状态路由、异常分支。

Document Parser

PDF / 文本简历解析，生成 resume_text。

Resume Extraction Agent

抽取 candidate_profile，支持规则与可选 LLM。

JD Extraction Agent

抽取 job_profile 和岗位要求结构。

Validator

Pydantic Schema 校验；失败时有限重试或失败报告。

Candidate Analyst

总结候选人优势、短板、信息缺口。

Job Analyst

提炼岗位重点、评估维度和优先级。

Memory Agent

检索历史人工反馈，形成 feedback summaries。

Rubric Selector

选择校招、社招、实习等评分轨道。

Matcher

计算 match_score，输出匹配明细与证据引用。

Risk Evaluator

预测 needs_human_review 风险，输出 risk_score。

Reporting Agent

汇总匹配和风险，形成最终建议。

Report Writer

生成 Markdown 报告，包含 Agent 协作摘要。

Human Review

人工确认、修订、补充反馈并写回 memory。

WorkflowState

candidate_profile、job_profile、agent_outputs、trace、report。

SQLite

runs、candidates、jobs、traces、reports、reviews、batches。

Feedback Memory

人工复核反馈沉淀，供 Memory Agent 检索。

Trace

记录每个节点执行摘要和关键元信息。

Replay

保存输入与观测结果，用于回放和回归验证。

Evaluation

报告质量、抽取评估、批量评估质量检查。

讲解口径：这不是 Agent 自由聊天系统，而是中心调度型多 Agent workflow。 Supervisor 负责编排，Specialist Agents 负责语义分析与汇总，Validator、Matcher、 Risk Evaluator 等确定性节点负责稳定约束和可复现计算。
